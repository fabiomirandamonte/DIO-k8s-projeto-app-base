📌 Descrição do Projeto

Projeto tem como objetivo a aplicação prática de conceitos fundamentais de conteinerização e orquestração utilizando Docker e Kubernetes.

A atividade consiste na construção e execução de uma aplicação em ambiente conteinerizado, seguida da sua implantação em um cluster Kubernetes. Durante o processo, são abordados conceitos como criação de imagens, definição de deployments, gerenciamento de pods e exposição de serviços.

Além disso, o projeto reforça boas práticas de organização, versionamento de código com Git e integração com repositórios remotos no GitHub.

O principal objetivo é simular um fluxo real de desenvolvimento e deploy, proporcionando uma visão prática de como aplicações modernas são estruturadas e disponibilizadas em ambientes escaláveis.
